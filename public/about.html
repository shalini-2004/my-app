About Our Book Review Web App

Our Book Review Web App allows users to search for books, submit and view reviews, and see recent review details. It features an engaging UI with a background video and leverages the Google Books API for book data, combined with a backend to manage user reviews.

Key Features

Book Search and Display:

Search Functionality: Users can search for books using the Google Books API by entering a book title or keyword. This feature supports both pressing the Enter key and clicking the search button.

Book Details: The search results display a list of books, including their titles, authors, descriptions, and cover images.

Review Submission:

Rating and Comment: Users can submit reviews for each book, including a rating (1 to 5 stars) and a textual comment.

Dynamic Review Update: Upon submitting a review, the app updates the review list and displays the most recent review prominently.

User Interface:

Background Video: A visually engaging background video enhances the user experience on the home page.

Responsive Design: The app layout is designed to be responsive and user-friendly, providing a seamless experience across different devices.

API Integration:

Google Books API: Utilizes the Google Books API to fetch book data based on user search queries.

Backend Integration: Interacts with a backend service to handle review submissions and fetch reviews for specific books.

Component-Based Architecture:

Modular Design: The app is structured into reusable components, such as Navbar, BookList, and Card, making it easier to manage and maintain.

Separation of Concerns: Different functionalities are handled by distinct components, improving code organization and readability.

State Management:

React Hooks: Uses React hooks (useState) to manage component state, including search queries, book data, review details, and visibility of review sections.

Utilizes useState hook in functional components or state in class components.
